Les Commandes GIT
Commandes Description
Git config On l’utilise pour configurer les préférences de l’utilisateur : son mail, l’algorithme utilisé pour diff, le nom d’utilisateur et le format de fichier etc.
Git init Cette commande est utilisée pour créer un nouveau dépôt GIT .
Git add La commande git add peut être utilisée pour ajouter des fichiers à l’index.
Clone git La commande git clone est utilisée pour la vérification des dépôts.
Git commit La commande git commit permet de valider les modifications apportées au HEAD.
Git status La commande git status affiche la liste des fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés.
Git push Git push est une autre commandes GIT de base. Un simple push envoie les modifications locales apportées à la branche principale associée.
Git checkout La commande git checkout peut être utilisée pour créer des branches ou pour basculer entre elles.
Git remote Cette commande remote permet à un utilisateur de se connecter à un dépôt distant.
Branche git La commande git branch peut être utilisée pour répertorier, créer ou supprimer des branches.
Git pull Pour fusionner toutes les modifications présentes sur le dépôt distant dans le répertoire de travail local.
Git merge La commande git merge est utilisée pour fusionner une branche dans la branche active.
Git diff La commande git diff permet de lister les conflits.
Git tag Le marquage est utilisé pour marquer des commits spécifiques avec des poignées simples.
Git log L’ exécution de cette commande génère le log d’une branche.
Git reset Pour réinitialiser l’index et le répertoire de travail à l’état du dernier commit.
Git rm Git rm peut être utilisé pour supprimer des fichiers de l’index et du répertoire de travail.
Git stash git stash aide à enregistrer les changements qui ne doivent pas être commit immédiatement.
Git show Pour afficher des informations sur tout fichier git.
Git fetch Git fetch permet à un utilisateur d’extraire tous les fichiers du dépôt distant qui ne sont pas actuellement dans le répertoire de travail local.
Git ls-tree Pour afficher un fichier arborescent avec le nom et le mode de chaque élément, et la valeur SHA-1 du blob.
Git cat-file À l’aide de la valeur SHA-1, affichez le type d’un fichier à l’aide de la commande git cat-file.
Git grep Git grep permet à un utilisateur de rechercher dans les arbres de contenu des expressions et / ou des mots.
Gitk Gitk est l’interface graphique du dépôt local.
Git instaweb Avec la commande git instaweb , un serveur Web peut être exécuté par interface avec le dépôt local. Qui redirige directement vers un serveur web.
Git gc Pour optimiser le dépôt en supprimant les fichiers inutiles et les optimiser.
git archive La commande git archive permet à un utilisateur de créer un fichier zip ou tar contenant les composants d’un arbre du dépôt.
Git prune Via la commande git prune , les fichiers qui n’ont pas de pointeurs entrants seront supprimés.
Git fsck Pour effectuer une vérification d’intégrité du système de fichiers git.
Git rebase La commande git rebase est utilisée pour la réapplication des commits sur une autre branche.
